# Sales Data Analysis using Python

This project analyzes a sales dataset using Python, Pandas, and Matplotlib to extract business insights and visualize trends.

## Features
- Dataset exploration and summary statistics
- Revenue and profit analysis
- Category and regional sales visualization
- Monthly revenue trend analysis
- Top products and top customers identification
- Quantity distribution analysis

## Technologies Used
- Python
- Pandas
- Matplotlib

## Visualizations
The project generates:
- Revenue by Category
- Profit by Region
- Monthly Revenue Trend
- Top Products by Revenue
- Quantity Distribution
- Sales by Region

A processed dataset is saved as:
processed_sales_data.csv
